# Quiz Federico

Sito statico per GitHub Pages.

## Come aggiungere foto

Metti le immagini nella cartella `images` usando questi nomi:

- `q1-a.jpg`, `q1-b.jpg`, `q1-c.jpg`, `q1-d.jpg`
- `q2-a.jpg`, `q2-b.jpg`, ecc.

Se un'immagine manca, il sito mostra automaticamente il placeholder.

## File principali

- `index.html` = struttura del sito
- `style.css` = grafica
- `script.js` = domande e logica del quiz
- `images/` = immagini delle risposte
